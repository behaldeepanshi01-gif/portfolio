## About

Interactive portfolio website built with vanilla HTML, CSS, and JavaScript. Features animated metrics, timeline-based experience section, skills grid, and project showcase.

## Sections

- **Hero** — Animated intro with key metrics (350 daily covers, 500+ CRM records, 92% satisfaction)
- **About** — Professional summary with highlight cards
- **Experience** — Interactive timeline (Bungalow NYC, Hilton Conrad, Virginia Tech, Radisson Blu)
- **Skills** — Four-category grid (Data & Analytics, CRM & Operations, Hospitality, Leadership)
- **Projects** — Featured analytics work with GitHub links
- **Education** — Academic background (McDaniel, Virginia Tech, IHM)
- **Contact** — Direct contact information

## How to View

Visit the live site via GitHub Pages, or run locally:

```bash
git clone https://github.com/behaldeepanshi01-gif/portfolio.git
cd portfolio
# Open index.html in any browser
```
